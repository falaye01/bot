# subhostbot

Professional VTU and WhatsApp Automation business website.

## Features
- Home Page with Hero, Services, and Contact sections.
- Meta-compliant Privacy Policy and Terms of Service.
- Responsive design with Tailwind CSS.
- WhatsApp integration.

## Tech Stack
- React
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React

## Deployment
This project is ready for deployment on Vercel or Netlify.
